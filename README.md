# A lightweight CLI based In Time Calculator for [greytHR](https://www.greythr.com)

**Installation**
```
git clone https://github.com/z0h4n/itc-greythr-cli.git
cd itc-greythr-cli
npm install
```

**Usage**
```
npm start
```
